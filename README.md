# Setting up the frontend

## Packages

    npm install

## .env file

Create a .env file with the following:

    REACT_APP_SOCKET_URL=http://localhost:8080
    REACT_APP_API_URL=http://localhost:8000/
    REACT_APP_SERVERS=4000, 4020, 4040, 4060, 4080

# Starting the frontend

    npm start
